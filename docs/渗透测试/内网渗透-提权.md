# **7月8日 内网提权**

### **什么是提权**

提权： 把低权限变为高权限

windows :  zhangsan  -> administrator -> system

linux : www-data  -> root 

### **提权的分类**

利用方式的不同：

本地提取：在一个低权限用户的前提下，通过漏洞提权至高权限用户

远程提权： 攻击者直接通过漏洞攻击获取最高权限用户

漏洞的不同：

操作系统提权：

windows：ms08067 ms1105

linux: dirty cow

应用程序提权：

mysql，mssql，IE，phpstudy

### **提权的思路**

提权的前提是目标系统存在漏洞或配置缺陷

- 基本信息收集

- 扫描系统、服务漏洞
- 寻找应用程序漏洞
